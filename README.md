# 🧁 Rincón Pastelero

Una página web dedicada a la repostería, recetas de pasteles y consejos culinarios. Proyecto desarrollado con **HTML**, **CSS** y **Bootstrap**.

## 📋 Descripción

Rincón Pastelero es un sitio interactivo donde los usuarios pueden:
- Explorar recetas de pasteles variadas
- Conocer detalles específicos de cada receta
- Conectar con una comunidad de pasteleros aficionados
- Acceder a consejos y trucos de repostería

## 🏗️ Estructura del Proyecto

```
rinconPastelero/
├── index.html              # Página principal con navbar, footer y algunas recetas
├── pages/
│   ├── recetas.html        # Página con todas las recetas disponibles
│   ├── detalle-receta.html # Página de detalle de cada receta
│   ├── comunidad.html      # Página de comunidad
│   └── 404.html            # Página de error 404
├── css/
│   └── style.css           # Estilos personalizados
├── img/                    # Carpeta con imágenes del proyecto
└── README.md               # Este archivo
```

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura del contenido
- **CSS3** - Estilos personalizados
- **Bootstrap 5** - Framework CSS para diseño responsivo

## 📄 Páginas del Proyecto

### 🏠 Index (Página Principal)
- Barra de navegación con menú de acceso
- Selección de recetas destacadas
- Sección de comunidad
- Pie de página con información de contacto

### 🍰 Recetas
- Listado completo de todas las recetas disponibles
- Tarjetas interactivas
- Enlaces para acceder al detalle de cada receta

### 📖 Detalle de Receta
- Información completa de la receta seleccionada
- Ingredientes y modo de preparación
- Tiempo de cocción y dificultad
- Consejos y variaciones

### ⚠️ Página 404
- Página de error para rutas no encontradas
- Navegación de retorno al inicio

## 🚀 Cómo Usar

1. Clonar el repositorio
2. Abrir `index.html` en tu navegador
3. Navegar por las diferentes secciones usando el menú

## 📱 Responsive Design

El proyecto está optimizado para verse correctamente en:
- 📱 Dispositivos móviles
- 📱 Tablets
- 💻 Computadoras de escritorio

Gracias a Bootstrap, la página se adapta automáticamente al tamaño de la pantalla.
